# Умная теплица

Проект автоматизации теплицы на ESP32.

Система собирает данные с датчиков температуры, влажности воздуха и влажности почвы, управляет поливом по 6 зонам и проветриванием через два привода. Настройки, расписания и ручные команды передаются через MQTT.

В проект входит:

- прошивка ESP32 в стиле Arduino;
- MQTT-контракт для обмена данными;
- Wokwi-схема и тестовые SHT30-чипы;
- мобильное клиентское приложение Flutter для Android.

Клиентское приложение позволяет подключиться к MQTT, видеть состояние теплицы, управлять поливом и проветриванием, задавать расписания и пороги автоматики.

APK лежит в корне проекта:

```text
SmartGreenhouse.apk
```

Исходники приложения:

```text
SmartGreenhouseApp/
```
